# TB6612-4路电机驱动

- 上传人：GUO
- 工程文件用【立创专业版打开】
- 集成了两个TB6612芯片直接驱动四路电机，用于低速的直流电机控制非常方便
- 安装孔的直径是3mm(M3螺丝)
