# STM32F103C8T6-V2.0最小系统板

- 工程文件用【立创专业版打开】
- 该模块是仿照市面上的STM32F103C8T6最小系统板子的改造升级版
- 把大部分贴片电阻和电容替换为0805封装，更加容易焊接
- 两边排针的距离刚好可以安装进面包板
- 该最小系统板有两个USB接口，一个用于芯片USB通信，另外一个作为USB转串口，可以直插进行串口调试不必再外接串口模块
